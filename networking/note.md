## multicast
    - method  of sending data from  one source to multiple receivers at once(only to those who join a multicast group)
    - IP range: 224.0.0.0 - 239.255.255.255
    - layer3(network layer)
## NetBios
    - API that allows applications on a local are network(LAN) to comunicate
        - UDP 137	NetBIOS Name Service	
        - UDP 138	NetBIOS Datagram Service	
        - TCP 139	NetBIOS Session Service
